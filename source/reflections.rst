Reflections
===========

How to Add Reflection PDFs
--------------------------

1. Put your PDF files in ``source/_static/pdfs/``.
2. Use the exact filenames shown below, or update the ``src`` paths.
3. Rebuild docs with ``build-pages.bat``.

General Education Reflection
----------------------------

Use this file path:
``source/_static/pdfs/general_education_reflection.pdf``

.. raw:: html

   <iframe
     class="pdf-viewer"
     src="_static/pdfs/general_education_reflection.pdf"
     title="General Education Reflection PDF"
   ></iframe>

If the embed does not load in your browser, open the file directly:
`General Education Reflection PDF <_static/pdfs/general_education_reflection.pdf>`_
